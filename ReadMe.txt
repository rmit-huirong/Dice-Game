Game rules:

1)	Player name is text; player ID is numerical; points is integer.
2)	You must input valid player information to add a specific player.
3)	You must input a valid player ID to remove a specific player.
4)	You must add at least a player before you can place bet.
5)	You must place a valid bet (i.e. sufficient and numerical) before you can roll a selected player.
6)	A player can only roll for one time in each round and result will be recorded. 
7)	You must roll at least one player before you can roll house.
8)	After rolling the house, all players' bets and roll results will be reset.
